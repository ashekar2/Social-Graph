# Social Graph
A social networking website that aims to create an environment for people to connect in a real-world setting. 

Interested in someone? Follow them! 
Want to tell someone how you feel about them? Send them a Poke!
Gather people, make those road-trip plans. Create Groups!

Requirements:
1) Neo4j Graph database
2) Flask
3) Python 2.7 or greater

Contact me for more information on it's usage and application.
